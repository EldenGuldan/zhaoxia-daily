# 飞书多维表格配置 - 朝霞 AI 日报

## 字段结构

| 序号 | 字段名 | 字段类型 | 配置说明 |
|------|--------|----------|----------|
| 1 | 标题 | 文本 | 默认 |
| 2 | 摘要 | 文本 | 多行文本 |
| 3 | 链接 | 超链接 | 默认 |
| 4 | 来源 | 文本 | 默认 |
| 5 | 来源类型 | 单选 | 选项：官方博客、开源社区、产品发布、社交媒体、科技媒体、AI社区 |
| 6 | 类别 | 单选 | **🛠️ AI工具** / **📰 行业新闻** / **🔥 热点话题** |
| 7 | 标签 | 多选 | 可选标签：大模型、AI绘画、AI编程、视频生成、AI音乐、AI搜索、开源模型、Agent、新品发布、社区热点 |
| 8 | 发布时间 | 日期 | 日期时间格式 |
| 9 | 点赞数 | 数字 | 整数 |
| 10 | 阅读量 | 数字 | 整数 |
| 11 | 评论数 | 数字 | 整数 |
| 12 | 互动量 | 数字 | 整数，可设置公式：点赞数+评论数 |

## 配置参数

```python
FEISHU_CONFIG = {
    "app_token": "YOUR_APP_TOKEN",  # 从飞书多维表格 URL 获取
    "table_id": "YOUR_TABLE_ID",    # 从数据表链接获取
    "view_id": None,
}
```
